# 打包与安装

## 打包

### Windows 上打包 exe

进入项目根目录用管理员模式启动cmd，运行：

```bash
pnpm build
```

打包完成后，产物在 `dist/artifacts/` 目录下，会得到两种 exe：

| 文件名示例 | 类型 | 说明 |
|---|---|---|
| `Biu-1.16.0-win-setup-x64.exe` | NSIS 安装包 | 有安装向导，会注册到系统"应用列表"，支持从控制面板卸载 |
| `Biu-1.16.0-win-portable-x64.exe` | 便携版 | 单文件，双击直接运行，无需安装 |

> arm64 机器同理，产物文件名中 arch 会是 `arm64`。

---

### macOS 上打包 dmg

在 Mac 机器上，进入项目根目录，运行：

```bash
pnpm build
```

打包完成后，产物在 `dist/artifacts/` 目录下：

| 文件名示例 | 类型 |
|---|---|
| `Biu-1.16.0-mac-x64.dmg` | Intel Mac 安装镜像 |
| `Biu-1.16.0-mac-arm64.dmg` | Apple Silicon 安装镜像 |
| `Biu-1.16.0-mac-x64.zip` | ZIP 压缩包（备用） |

---

## 安装与运行

### Windows

**NSIS 安装包（推荐）**

1. 双击 `Biu-xxx-win-setup-x64.exe`
2. 按安装向导提示选择安装目录，点击"安装"
3. 安装完成后桌面会出现 Biu 快捷方式，双击运行

效果和网易云音乐一样：有安装过程、有开始菜单/桌面图标、可从控制面板卸载。

**便携版**

直接双击 `Biu-xxx-win-portable-x64.exe` 即可运行，无需安装，适合不想动系统的场景。

---

### macOS

1. 双击 `Biu-xxx-mac-arm64.dmg`（Apple Silicon）或 `Biu-xxx-mac-x64.dmg`（Intel）
2. 弹出一个窗口，里面有 Biu 图标和 Applications 文件夹快捷方式
3. 将 Biu 图标**拖拽**到 Applications 文件夹上
4. 打开启动台或 Applications 文件夹，找到 Biu，双击运行

效果和 Mac 版网易云音乐一样：拖拽安装，装到应用程序文件夹。

**首次运行提示"已损坏"或"无法验证开发者"**

由于未做苹果公证（`notarize: false`），首次打开会被 Gatekeeper 拦截。解决方法：

```bash
# 在终端执行，移除隔离属性
xattr -cr /Applications/Biu.app
```

或者手动操作：系统设置 → 隐私与安全性 → 往下滚动 → 点击"仍要打开"。

之后每次正常双击即可打开，不会再提示。
