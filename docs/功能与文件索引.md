# Biu 功能与文件索引

> 本文档按功能模块列出涉及的源码文件，便于快速定位代码、分析逻辑、修复问题。
>
> 路径均相对于项目根目录。文件角色说明：
> - **页面/组件** — React UI，渲染进程
> - **状态** — Zustand store，渲染进程全局状态
> - **IPC 主进程** — Electron 主进程处理器
> - **服务/API** — Bilibili 或第三方 API 请求封装
> - **类型** — TypeScript 类型定义

## 补充：跨功能基础文件

| 文件 | 用途 |
|------|------|
| `electron/main.ts` | Electron 主进程入口，初始化所有 IPC 处理器 |
| `electron/preload.ts` | Preload 脚本，将所有 IPC API 暴露给渲染进程 |
| `electron/ipc/channel.ts` | 全部 IPC 通道名称的单一来源 |
| `src/routes.tsx` | 路由配置，所有页面路径定义 |
